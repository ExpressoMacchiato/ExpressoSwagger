# Requisiti ExpressoSwagger

1. **Stack UI**: React con Tailwind CSS.
2. **Sistema di Configurazione**: Gestione dedicata per definire correttamente le configurazioni (flessibile e potente).
3. **UI Layout**: Struttura a tre colonne, animata il giusto, con Tailwind.
4. **Developer Experience**: Impostazioni lato developer il più semplici e intuitive possibili.

## Focus Tecnico
- Supporto nativo per `withCredentials` (Auth custom/cookie).
- Approccio programmatico (niente decoratori).
- Architettura a libreria.
